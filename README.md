**💡 555 Timer LED Sequencer | PCB Layout Design using Autodesk EAGLE**

**📖 Overview**

Designed a single-layer PCB for a 555 Timer-based LED Sequencer using Autodesk EAGLE from a provided schematic. The project involved component placement, manual trace routing, and PCB layout development to create a manufacturable design. The circuit generates sequential LED illumination using a NE555 Timer and CD4017 Decade Counter.

**✨ Features**

- ✅ Single-layer PCB layout
- ✅ Manual component placement
- ✅ Manual trace routing
- ✅ Sequential LED indication
- ✅ Through-hole PCB design
- ✅ Compact PCB layout


**⚙️ Working Principle**

- The **NE555 Timer** operates in astable mode to generate clock pulses.
- The **CD4017 Decade Counter** receives these pulses and activates one LED output at a time.
- Each clock pulse advances the sequence, creating a running LED effect.
- The sequence repeats continuously, producing a visually appealing LED pattern.
